# Hotel Orders & Customer Database (SQL)

## Project Overview
This project involves designing and querying a relational database to manage hotel orders and customer data efficiently.

## Objectives
- Design structured database tables
- Store and manage customer and order data
- Extract insights using SQL queries

## Tools & Concepts
- SQL (SQL Server)
- Clauses
- Aggregations and filtering

## Database Features
- Customer and order management
- Revenue analysis queries
- Customer behavior analysis

## Conclusion
This project demonstrates strong SQL querying and relational database design skills.
